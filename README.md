# Estate Agency Transaction System

A full-stack application to manage the post-agreement lifecycle of real estate
transactions and automate commission distribution between the agency and agents.

## Stack

**Backend:** NestJS · TypeScript · MongoDB Atlas · Mongoose · Jest
**Frontend:** Nuxt 3 · Pinia · Tailwind CSS
**Deployment:** Railway (API) · Vercel (Web) · MongoDB Atlas (DB)
